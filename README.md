📚 Student Study Planner

A simple and efficient Student Study Planner web application designed to help students organize their study schedules, manage tasks, and improve productivity.

---

🚀 Features

- 📅 Add and manage study tasks
- ✅ Mark tasks as completed
- 🗂 Organize subjects and schedules
- ⏰ Plan daily study routines
- 💻 Simple and user-friendly interface

---

🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Node.js

---

📂 Project Structure

- "index.html" – Main webpage
- "style.css" – Styling
- "index.js" – Frontend logic
- "server.js" – Backend server
- "package.json" – Project dependencies

---

▶️ How to Run the Project

1. Clone the repository
   
   git clone https://github.com/your-username/student-study-planner.git

2. Open the project folder

3. Install dependencies
   
   npm install

4. Run the server
   
   node server.js

5. Open in browser
   
   http://localhost:3000
